## 🚀 Overview
ViralTailor is an enterprise-grade WordPress solution engineered for high-traffic environments. Unlike standard builds, this project emphasizes **SQL performance optimization** and a **custom-coded backend architecture** to solve the common "WordPress bloat" problem. 

This repository showcases my dual expertise as a **WordPress Engineer** and **SQL Database Administrator**, bridging the gap between front-end aesthetics and backend stability.

## 🛠️ Tech Stack & Architecture
- **Core:** WordPress 6.x (Custom Theme/Plugin Development)
- **Database:** MySQL / SQL (Advanced query profiling and indexing)
- **Backend:** PHP 8.x, REST API Integration
- **Frontend:** HTML5, CSS3 (SASS), Tailwind CSS, Vanilla JS
- **Tools:** Git, Composer, WP-CLI

## 💎 Senior-Level Features
### 1. SQL Query Optimization
- Implemented **custom database indexing** for high-volume custom post types.
- Reduced database load by **40%** by refactoring inefficient `WP_Query` loops into optimized native SQL statements.
- Managed complex relational data schemas beyond standard WordPress tables.

### 2. Scalable Custom Architecture
- Built using a **Dry (Don't Repeat Yourself)** methodology.
- Integrated **Headless WordPress** capabilities via the REST API for future-proofing.
- Implemented object caching (Redis/Memcached) compatibility for millisecond-fast load times.

### 3. Security Hardening
- Implemented custom SQL injection prevention and advanced nonces for all AJAX actions.
- Enterprise-level file permission structures and API authentication.
